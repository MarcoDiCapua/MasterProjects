Per Phyton installare i requirements con
pip install -r "requiremens.txt"

Per far funzionare inserire le matrici che si trovano nella la cartella \progetto\Matrices dentro la cartella \progetto\Nome_cartella\matrices

Per le cartelle Matlab e Matlab linux sono state usate le matrici 1-6
Per le cartelle Python e Python linux sono state usate le matrici 1-4
